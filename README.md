# Vibekit

A fork of spec-kit with improvements for better usability.

## Features

- Integrated spec-kit as submodule
- Custom modifications for enhanced spec-based workflows

## Getting Started

Clone the repository and initialize submodules:

```bash
git clone https://github.com/congvmit/vibekit.git
cd vibekit
git submodule update --init --recursive
```
